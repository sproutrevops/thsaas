TH SAAS — Platform Overview

TH SAAS is an internal data platform that connects Google Sheets and HubSpot CRM to give your team a unified view of MRR, headcount, and client movement across products and partners.

This version runs on a Node.js/Express server with Google OAuth authentication. Sign in via Google; access is controlled by the Allowed Emails and Allowed Domain settings configured in .env.

Pages Reference

▣ Dashboard

Home screen with summary cards (Sheet Tabs, HubSpot Companies, Server Status), connection health indicators, and quick action buttons.

📦 Per Product Movement (PPM)

Main data table. Displays every client–product combination with monthly HC (headcount) and computed MRR.

Three sheets required:

  • PPM Sheet — client list with Partner, Client Name, Product, Application, Start Date, Renewal Date, MRR.
  • HC Sheet — monthly headcount per client (columns: HC YYYY-MM). HC > 0 = active.
  • Unit Price Sheet — per-seat pricing; used to compute MRR = HC × unit price.

Features: partner/client/application filters, sticky columns for Partner + Client Name + dates, group rows by partner, stats bar (Total MRR, HC, Active Clients, Partners).

📈 SaaS Metrics

MRR waterfall (Beginning → New Business → Expansion → Contraction → Lost → Ending) computed per month from PPM data. Load PPM first. Exportable as CSV.

🏢 Company Movement

High-level MRR movement overview rolled up across all products and partners. Uses PPM data — load PPM first.

📊 Google Sheets

Raw data browser — lists all spreadsheet tabs; click any tab to view its contents as a table.

🔶 HubSpot CRM

Displays company records from HubSpot. Requires a Private App token in .env (HUBSPOT_ACCESS_TOKEN).

⚙️ Settings

  • Spreadsheet URL or ID — paste the full URL or just the ID (between /d/ and /edit).
  • HubSpot Access Token — Private App token from HubSpot → Settings → Integrations → Private Apps.
  • Connection Status — test live connectivity to both services.

Data Logic & Calculations

Active vs Inactive

A client is active when HC > 0 in the HC sheet for that month. HC = 0 or blank = inactive / churned.

Computed MRR

MRR (month) = HC (month) × Unit Price

SaaS Metrics categories

  • New Business — HC was 0 last month, HC > 0 this month.
  • Lost / Churn — HC was > 0 last month, HC = 0 this month.
  • Expansion — active both months, MRR increased.
  • Contraction — active both months, MRR decreased.

Troubleshooting

Symptom Likely cause & fix
Dashboard shows Not configured Set GOOGLE_SHEETS_ID in Settings and save.
PPM loads no data Select the correct sheet from the PPM Sheet dropdown, then click Load.
SaaS Metrics shows nothing Load Per Product Movement first, then return to SaaS Metrics and click Refresh.
Stale data Click ↻ Refresh or ↻ Reload Tabs on the relevant page.
